== Description ==
Premium WooCommerce B2B & Wholesale Plugin for advanced business management.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/b2b-commerce-pro` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure settings via the B2B Commerce Pro menu.

== Features ==
* Advanced user roles and approval workflow
* Sophisticated pricing system
* Product and category restrictions
* Modern admin panel
* Responsive frontend dashboard
* Credit, tax, and payment terms
* REST API and integrations
* Translation ready
* Developer hooks for extensibility
* Performance optimized
* Uninstall removes all plugin data
